# Verification documentation

The public repository keeps the reusable testing strategy and all executable test sources. Device-specific acceptance receipts, private GitHub run URLs, hosted-minute ledgers, and maintainer session journals are intentionally omitted from the history-free public snapshot.

Some implemented milestone specs describe those private evidence files because they record the original acceptance contract. Those references are historical context, not missing setup required to build or test a public clone. Contributors should attach fresh verification evidence to their issue or pull request and use the commands in [testing-strategy.md](testing-strategy.md).
